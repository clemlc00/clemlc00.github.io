Propuesta Grupal

Grupo conformado por: María Jose Bascuñan, Manuel Diaz, Emiliano Leixlard y Clemente León

Propuesta

Nuestra propuesta de diseño es la creación de una familia de sillas las cuales estan diseñadas para interior, con el fin de que perduren en el tiempo. El tema nativo para nosotros es parte fundamental del proyecto, por lo que, utilizaremos madera para la creación de esta familia y que comparta el diseño generativo.

Como proceso de creación queremos utilizar el fresado en CNC.

Fresado CNC.

¿Qué es?

Una máquina CNC es un dispositivo controlado por un ordenador que puede cortar, tallar y fresar diseños programados previamente. Y hacerlo sobre diferentes materiales como madera, MDF, plásticos, espumas y metales, etc.
